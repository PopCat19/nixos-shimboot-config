# Context

- `generate-readme.sh` — Concatenate readme fragments into README.md with boundary markers
- `readme-to-fragments.sh` — Extract README.md sections back into individual fragment files
- `check-readme-drift.sh` — Detect stale commit citations in README.md
- `check-refs.sh` — Validate file references and line-number citations in markdown documentation
- `readme.sh` — Unified entry point for README workflow (generate, validate, drift check)
